Aqua-Checkin App

This app will be used to account for all Aqua employees in case of an emergency evacuation.
The app can be accessed from any internet connected computer, phone or tablet.

How to take ROLL CALL using Aqua-Checkin App (Loves Park location only)

  • First, make sure employee/guests are in designated area BEFORE taking roll call for your area.
  • Begin typing in department name to see all in your area or first/last name
  • Tap IN button once employee is verified as being present.
  • That’s it!

  • Tap OUT button if person is mistakenly checked in

How to view Aqua-Checkin App Report

  • Tap Report button
  • The report will ONLY show ROLL CALL results for current date.

How to add Guests

  • Guest that you know will be visiting Aqua in the future must be added manually to the Aqua-Checkin database
  • First, check to see if they are already in database by searching for guest name on home page searchbox
  • If they are not in the database, tap "Add Guests" button to add guest name

How to setup Aqua-Checkin App

Computer/Laptop

  • Open https://aquacheckin.github.io from the Chrome web browser

    • Either bookmark webpage or follow steps below to save as desktop shortcut.

  • Click the Chrome menu on the browser toolbar
  • Select More Tools
  • Select Createshortcut
  • Click Create
  • You should have a new shortcut on your desktop titled: Aqua-Checkin

iPhone

Best viewed if phone is in landscape mode.
  • Open https://aquacheckin.github.io from Safari web browser

    Either bookmark webpage or follow steps below to save as homescreen shortcut.

    See Picture Steps below on how to add to iPhones home screen

  • Tap arrow icon on bottom of iphone (see circled icon)


  • Tap Add to Home Screen (see circled icon)


  • Tap Add (see circled icon)


  • You should now see the Aqua-Checkin icon on your iPhones home screen! (see image below)


Android

Best viewed if phone is in landscape mode.
  • Open https://aquacheckin.github.io from the Chrome web browser

    • Either bookmark webpage or follow steps below to save as homescreen shortcut.

  • Tap on the Menu button. The looks of the Menu button will depend on your device; usually it’s an app button that looks like
